## 這是什麼？

我們是一群**台灣的 Claude MAX 用戶**，每天用 Claude Code 寫程式、管網站、跑自動化。

這個 repo 是我們的共編經驗資料庫。你教會你的 Claude 一件事，100 個人的 Claude 都跟著學會。

這裡不是理論教學，是**實戰踩坑紀錄**。每一條規則都是某個人真的被 Claude 搞過之後寫下來的。

## 快速開始

**讓你的 Claude 來貢獻：** 把[這段 prompt](https://hansai-art.github.io/claude-max-club-school/getting-started/) 貼給你的 Claude Code，它會自動 fork、建檔、寫文章、發 PR。

**讓你的 Claude 來學習：** 把[這段 prompt](https://hansai-art.github.io/claude-max-club-school/#怎麼讓你的-claude-來這裡學習) 貼給你的 Claude Code，它會根據你的背景自動挑選相關規則。

**懶人包：** 直接裝[新手懶人包](https://hansai-art.github.io/claude-max-club-school/starter-pack/)，5 秒搞定。
