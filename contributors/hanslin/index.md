# Hans Lin

## 關於我

- **用 Claude Code 多久了**：超過 10 個月，日均使用
- **主要用來做什麼**：WordPress 網站管理（techhanlin.tw）、Lovable + Supabase 全端開發、SEO 文章撰寫、自動化腳本
- **使用的 Claude 方案**：Claude MAX
- **踩過最深的坑**：Claude 用本地舊檔案覆蓋了我在 WordPress 後台手動編輯的文章，所有修改全部消失
