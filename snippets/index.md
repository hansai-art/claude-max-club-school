## 使用方式

1. 找到你需要的規則
2. 複製 code block 裡的內容
3. 貼進你的 `~/.claude/CLAUDE.md`（全域）或專案的 `CLAUDE.md`
4. 完成。你的 Claude 現在學會了這件事。
